인증서 생성 툴

OpenSSL
여기서는 OpenSSL 을 사용한다.
  • Easy-RSA
  • OpenSSL
  • CFSSL

생성 프로세스

키 생성

1

openssl genrsa -out ca.key 2048

인증서 서명 요청(CSR)

1

openssl req -new -key ca.key -subj "/CN=KUBERNETES-CA" -out ca.csr

인증서 서명

1

openssl x509 -req -in ca.csr -signkey ca.key -out ca.crt

CA(인증 기관) 생성

CA 키 생성

1

openssl genrsa -out ca.key 2048

CA 인증서 서명 요청(CSR)

1

openssl req -new -key ca.key -subj "/CN=KUBERNETES-CA" -out ca.csr

CA 인증서 서명

Note
-signkey is 'ca.key.
1

openssl x509 -req -in ca.csr -signkey ca.key -out ca.crt

Admin User 생성

Admin User 키 생성

1

openssl genrsa -out admin.key 2048

Admin User 인증서 서명 요청(CSR)

1

openssl req -new -key admin.key -subj "/CN=kube-admin/OU=system:masters" -out admin.csr

Admin User 인증서 서명

Note
-CA is 'ca.crt.
-CAkey is 'ca.key.
There’s no -signkey.
1

openssl x509 -req -in admin.csr –CA ca.crt -CAkey ca.key -out admin.crt

KUBE API SERVER

인증서

KUBE-API SERVER

키 생성

1

openssl genrsa -out apiserver.key 2048

CSR(인증서 서명 요청)

1
2
3
4
5

# No option
openssl req -new -key apiserrver.key -subj "/CN=kube-apiserver" -out apiserver.csr

# With options
openssl req -new -key apiserrver.key -subj "/CN=kube-apiserver" -out apiserver.csr -config openssl.cnf

openssl.cnf

CRT(인증서)

1
