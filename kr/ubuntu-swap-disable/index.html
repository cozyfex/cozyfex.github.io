컴퓨터에서 Swap 이란??

SWAP
메모리 안에서 데이터를 페이지나 세크먼트로 교체하는것이다.
스와핑은 메인 메모리보다 큰 다양한 데이터 파일들이나 실행 프로그램들을 화성화 하기 위한 유용한 기술이다.
더 자세한 사항은 여기로!
