## Go package manager
----

##### Initialize

<sub>Go init</sub>

```shell
go init mod [Package Name]
```

This command generate a `go.mod` file.

##### Generate requirements

<sub>Go requirements</sub>

```shell
go mod tidy
```

This command add requirements to `go.mod` and generate `go.sum`.
