# Kubernetes Software Versions


## Kubernetes Software Version Structure

`v1.10.1`

`v<Major>.<Minor>.<Patch>`

{{<admonition note Major true>}}

- Big change of the application.

{{</admonition>}}

{{<admonition note Minor true>}}

- Features
- Functionalities

{{</admonition>}}

{{<admonition note Patch true>}}

- Bug Fixes

{{</admonition>}}
